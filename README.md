PHP requirements
============= 
 * PHP >= 5.4.0
 * PHP extension `hash`
 * PHP extension `curl`
 * PHP extension `json`
 * PHP extension `pcre`
 * PHP extension `ctype`
 * PHP extension `rrd` (optional, for rrd graphs)
